#quick start

## Vue.js

###  node.js setup
https://nodejs.org/en

###
npm install -g @vue/cli

### creates vue project
```
vue create hello-world

or 

vue ui
```
### installs axios
```
npm install axios

### installs element-ui (for vue2)
```
npm install i element-ui -S

### installs element-plus (for vue3)
```
npm install element-plus --save
```
### Compiles and hot-reloads for development
```
npm run serve
```
